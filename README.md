# Earth Ocean Learning App

## Description
Interactive Angular web application for exploring oceans, seas and marine ecosystems.

The project provides structured educational content about marine environments, including biodiversity, climate and depth information.

## Technologies
- Angular
- TypeScript
- HTML / CSS

## Features
- Exploration of major oceans and seas
- Educational content about marine ecosystems
- Dynamic navigation between sections
- Component-based architecture

## Structure
The application uses multiple routes to organize educational content about oceans and seas.

Each section is separated into dedicated views for clarity and navigation.

## Purpose
This was my first hands-on Angular project.

It helped me learn component-based architecture, routing and structuring a frontend application.
